Sponge 설명문서¶

개요¶

오픈소스 Sponge 프로젝트의 공식 설명문서인 SpongeDocs에 오신 것을 환영합니다.

SpongeAPI는 새로운 Minecraft 용 플러그인 API입니다. 순정 바닐라 서버에서 (SpongeVanilla로) 또는 완전히 개조된 MinecraftForge 서버에서 (SpongeForge로) 구동할 수 있습니다. 여기서 받으세요.

무엇을 보아야 할지 모르시겠나요? 짧은 Introduction 와, 자주 묻는 질문, The Structure of the Sponge Project 에 대한 글을 포함하고 있는 Sponge 프로젝트에 대하여 절을 읽어보세요.

서버 주인들은 Creating a Server 에 있는 어떻게 Sponge 기반 서버를 설치, 운영, 유지하는지에 대한 저희의 조언과 설명을 읽어야 합니다.

Sponge 프로젝트에는 몇 가지 주요 구성 요소들이 있습니다. 여기서 프로젝트 구조에 대해 더 자세히 알아보거나 GitHub에 있는 요소들의 저장소들을 방문하십시오:

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore

주석

Sponge는 아직 개발되는 중입니다. SpongeForge와 SpongeVanilla의 베타 빌드들은 다운로드 할 수 있으나, 공식 릴리즈는 2016년 말에야 사용 가능할 것입니다.

감사의 말¶

SpongePowered 팀은 YourKit 팀이 완전한 기능을 갖춘 Java 프로파일러를 오픈 소스 프로젝트들에게 무료로 제공한 것에 감사드립니다. YourKit, LLC는 Java와 .NET 응용 프로그램들을 프로파일하기 위한 혁신적이고 지능적인 도구들인 YourKit Java Profiler 와 YourKit .NET Profiler 의 제작사입니다.
